Monologue Chat [Javascript | HTML | CSS]

This application allows a user to chat with themselves. Chat thread automatically scrolls to the bottom when new messages are added. If a user scrolls up to read earlier messages and new messages are added to the thread, a notification appears with a count of how many messages are waiting at the bottom of the thread, but auto scroll will not steal the screen from the user. Notification clears when user returns to the bottom of the thread. Messages can be removed after being sent. Messages stagger left and right so it feels slightly less lonely.
