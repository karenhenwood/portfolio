sessionStorage Registration [Javascript | HTML | CSS | sessionStorage]

This page works in conjunction with the sessionStorage Login page to simulate a user login and registration workflow. It rejects usernames that already exist in the known users, has simple password complexity requirments for new passwords, selectively clears only the bad password if a user doesn't meet complexity requirements, registers new users for use by the login page, and redirects users to the login page after addition
